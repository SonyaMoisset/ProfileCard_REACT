# Profile

## A React web application that renders a Profile Card

To see the project:
- Clone/download it
- Use ```npm install```
- Use ```npm start```

## This application includes examples of major features of React including:
- ES6 syntax
- Templating string
- Destructuring assignment
- Component-based architecture
- CSS Partials/Imports
